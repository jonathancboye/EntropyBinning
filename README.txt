To install on unix machine:
   	
	Type: make

To run the program:
   	
	Type: ./entropyCRsfs.exe <int> <inputFile>

	where <int> is a integer and <inputFile> is a name of a file
	formatted like the test file "p1data.txt" except it can
	have more tuples and genes.
